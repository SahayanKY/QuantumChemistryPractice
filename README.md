# はじめに
このリポジトリは量子化学の本に載っている問題の解答を保存することを目的としています。

Szabo
- 新しい量子化学　電子構造の理論入門
- Attila Szabo, Neil S. Ostlund 著
- 大野公男, 阪井健男, 望月祐志 訳
- 2006
- 東京大学出版会
